以下の仕様で、HTML/CSS/JSが1ファイルにまとまった宇宙パーティクルシミュレーションを作成してください。 ## 基本仕様 - ファイル名: cosmic-particle.html - HTML5 Canvas を使った全画面パーティクルシミュレーション - 完全オフライン動作（外部ライブラリ不使用） - レスポンシブ対応（リサイズ追従） ## シミュレーションモード（5種類） 1. 重力（Gravity）: パーティクルが中心に引き寄せられる 2. 渦（Vortex）: 渦巻き状に回転する 3. 花火（Fireworks）: クリック位置から爆発するように広がる 4. 銀河（Galaxy）: 回転する銀河のような動き 5. 波動（Wave）: 波のように揺れる ## インタラクション - マウス移動/タッチでパーティクルが反応 - クリック/タップでエフェクト発生 - モード切替ボタン（画面下部にピル型ボタン） ## デザイン - 背景: 暗い宇宙色（#080b12） - パーティクル: 発光感のあるグラデーション色 - UI: ダークテーマ、半透明ガラスモーフィズム - 左上に「← Back to Toolbox」リンク（href="index.html"） - パーティクル数: 500〜1000個程度（パフォーマンス考慮） ## 技術要件 - requestAnimationFrame でアニメーション - パーティクルプールで GC 負荷軽減 - Canvas 2D Context 使用
